# Finance Backend API

## Features
- User and Role Management
- Financial Records CRUD
- Filtering (date, category, type)
- Dashboard Summary APIs
- Role-Based Access Control
- Validation and Error Handling
- PostgreSQL + Prisma

## Setup
1. npm install
2. Configure .env
3. npx prisma migrate dev
4. node src/app.js

## API Base URL
http://localhost:5000
